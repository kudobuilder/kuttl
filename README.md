# kuttl
KUbernetes Test TooL (kuttl)

## Installation

Brew Install
```bash
brew install kuttl-cli
```

Krew Install
```bash
kubectl krew install kuttl
```

Go Install
```bash
cd $(mktemp -d) && go get github.com/kudobuilder/kuttl/cmd/kubectl-kuttl && cd -
```

## Usage

For usage, see

```bash
kubectl kuttl
```

or the documentation: http://kuttl.dev
